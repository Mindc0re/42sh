 cd [-L|-P] [dir]

 Options:
        -L	force symbolic links to be followed
        Tradutcion: Entrer dans le lien symbolic
        -P	use the physical directory structure without following symbolic
    	links
    	Traduction: Aller directement au dossier pointer par le lien symbolic

 cd nimporte_quoi dir:
 	- cd va chercher dans le pwd "nimporte_quoi" et le remplacer par "dir"
